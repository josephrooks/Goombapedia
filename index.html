Paper Mario (N64) · Tattle Checker

How to use

  1. In RetroArch with Paper Mario running and the save file you want to check loaded, open Quick Menu → Save State (or press your hotkey). You can do this even in RetroAchievements Hardcore Mode; you just cannot load states while Hardcore Mode is on.
  2. Find the .stateN file in your states/ folder (e.g. Paper Mario.state1) and upload it here. Save states only; .srm battery save files are not supported since the tattle data only exists in active memory, not in the save file itself.
  3. The page will show your overall progress, a chapter-by-chapter breakdown of every tracked enemy, and which ones you still need, with notes on missables and special conditions.
